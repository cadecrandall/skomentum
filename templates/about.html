Nocturne was built for HackMIT 2020, a hackathon put on by MIT that was done virtually this year due to the COVID-19 pandemic. Our team sought to create a program that could create a personalized piece of music, while only given a zip code and a date. {# TODO: remove date if we didn't implement that? #}

We created Nocturne using Flask, a lightweight web framework using Python to connect the frontend to the backend. This allowed for us to import our data into Python, generate music using the PyDub package, then display the location of the stars and play the music using HTML5 and Javascript.
